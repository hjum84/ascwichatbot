Storage Management
✅ To reduce storage usage:
  • Enable retention/auto-delete for chat history
  • Delete stale `authorized_users` whitelist rows
  • Archive or trim oversized chatbot content blobs
  • Run `VACUUM` (SQLite) / `VACUUM ANALYZE` (PostgreSQL)
🚀 To improve capacity accuracy:
  • Set `DB_MAX_SIZE_BYTES` to your actual Neon/Render plan limit
  • Review this page after plan upgrades/downgrades
  • Track dead/reclaimable growth and vacuum regularly
  • Use monthly archive exports for long-term retention
